CS Ed Week Task #3

Time to make something with code!

So here's the deal:

You need to code something digital and beautiful (or funny or emotional or intelligent or whatever you choose) for one of the two options below using Scratch.

Scratch is a very accessible block-based coding language - so you can use it even if you've never seen it before or usually write your code using a different language!

You'll submit your project using the link at the bottom of this page. However, you have a chance or some bonus points by sharing your project on whatever social media platform you'd like (TikTok, Instagram, Twitter, etc) and tagging with the hashtags:

  • #csEdWeek
  • #cs4all
  • #scratch
  • You can also link us with @CSforAllNYC and @Scratch.

Option 1: CS Heroes

Create a program that will educate the audience about someone who has made an important contribution to computing.

While you can pick anyone, we would recommend you think about those people often overlooked, like Evelyn Boyd Grandville, Jerry Lawson, Joy Boulamwini, Edie Windsor, Window Snyder, Lyndsey Scott, etc etc.

Option 2: CS for Everyone

It's no secret that we think everyone should learn computer science so they feel empowered to become creators of the technology we use every day. We want to know what YOU think would allow everyone to learn CS!

Imagining that the sky is the limit, create a program that informs the viewer of your ideas or plan to ensure everyone learns computer science.

Submit your project here!

(Submitting the project will also send you the link to the next task.)
